Dhabyan designs, builds, and operates high-efficiency mining and compute infrastructure with aligned, profit-sharing structures for institutional and high-net-worth clients.

- **Project 33**: Physical hosting model for direct hardware ownership

## Tech Stack

- **Frontend Framework**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **Icons**: Lucide React

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/DBYN-DHABYAN/DBYN-MAIN.git

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
